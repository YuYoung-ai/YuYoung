📋 AS 처리 보고서

현장 AS 처리 내용을 4단계로 입력하여 PPT 보고서를 자동 생성합니다.

🔑 접속 메뉴 허브에서 AS 보고서 선택 → 언어 선택 → 일반 비밀번호 입력
📌 전체 흐름
① 접수정보 → ② 사진첨부 → ③ 원인분석 → ④ 최종확인 → PPT 생성
1
AS 접수 정보 입력
필수 ▼
  • 병원명 *글자 입력 시 자동완성 드롭다운 표시 → 선택하면 S/N, 구독플랜, 영업담당자 자동 입력
  • 장비 S/N *병원 선택 시 자동 채워짐. 수동 입력도 가능
  • 제조일예) 2022년 11월
  • UI/FW/HP Ver예) 2.1.1 / 2.6 / 1.2.2
  • 발생 일자 *달력 팝업에서 선택
  • 담당자 *본인 이름 (PPT 서명란에 표시됨)
  • 요청사항예) 노즐 인식 불가 (토픽 한 줄 요약)

🔧 내부 수리 부품 (선택)

  • 부품명예) RFID Module A'ssy — 자동완성 23개 부품 지원
  • 교체 사유예) 노즐 인식 불가에 의한 교체
💡 팁 부품명 앞 글자 입력 시 자동완성 목록이 표시됩니다. 단가표가 함께 저장되어 있어 선택 즉시 단가가 채워집니다.
2
사진 / 영상 첨부
선택 ▼
  • 증상 사진최대 3장 (좌측) — 접수 당시 증상 화면·외관 촬영
  • 현상 사진최대 3장 (우측) — 현장 확인 내용 촬영
  • 동영상1개 — PPT에 재생 가능한 형태로 삽입
⚠️ 동영상 용량 동영상 포함 시 PPT 파일 크기가 크게 늘어납니다. 30초 이내 짧은 영상을 권장합니다.
💡 팁 이미지를 누르면 미리보기, X 버튼으로 삭제. 드래그 앤 드롭도 지원합니다.
3
고장 원인 분석
필수 ▼
  • 접수 증상 *태그 버튼 클릭 또는 직접 입력 (최대 40자)
  • 불량 원인 *태그 버튼 클릭 또는 직접 입력 (최대 32자)
  • 해결 방법 *태그 버튼 클릭 또는 직접 입력 (최대 32자)
  • 처리 의견엔지니어 처리 의견 (최대 45자)
  • 고객 피드백현장 고객 반응 기록 (최대 45자)
💡 빠른 입력 — 태그 버튼 자주 쓰는 증상/원인/해결 방법이 태그로 제공됩니다. 탭하면 즉시 입력되며, 이후 수정도 가능합니다.
노즐 인식 불가 USB 인식 불량 전원 불량 분사 불량
4
최종 확인 및 생성

❓ 자주 묻는 질문

현장에서 자주 발생하는 문제와 해결 방법을 모았습니다.

Q
오프라인 상태에서 앱이 작동하지 않아요
▼

인터넷이 연결된 상태에서 한 번 이상 접속해야 Service Worker가 캐시를 구성합니다.

  • 1️⃣인터넷 연결 상태에서 PWA에 접속
  • 2️⃣모든 도구를 한 번씩 열어서 캐시 적재
  • 3️⃣이후 오프라인에서 정상 작동
💡 현재 캐시 버전 baz-cs-v28 — 업데이트 시 자동으로 새 버전으로 교체됩니다.
Q
병원 자동완성에 병원이 안 나와요
▼
  • 🔤병원명 앞 글자부터 입력하세요 (공백 포함 검색 지원)
  • 📋DB에 없는 병원은 수동 입력 후 사용 가능
  • 🔄병원 추가 요청은 CS 팀장에게 연락하세요 (DB 업데이트 후 캐시 갱신 필요)
Q
구글 시트에 기록이 안 돼요
▼
  • 📶인터넷 연결 상태를 확인하세요 — 오프라인에서는 전송 불가
  • ⚙️Apps Script 배포 URL이 코드에 올바르게 설정되어 있는지 확인하세요
  • 🔒스크립트 액세스 권한(누구나 접근 가능)이 설정되어 있는지 확인하세요
ℹ️ 참고 실패한 기록은 자동 재전송되지 않습니다. 미전송 건은 수동으로 시트에 직접 입력하세요.
Q

  • 📲공유 버튼을 눌러 파일에 저장을 선택하세요 (iCloud Drive 또는 기기 내)
  • 📁저장 위치: 파일 앱 → 다운로드 폴더
  • 💬카카오 공유 버튼으로 카카오톡 채팅방에 파일 전송도 가능합니다
