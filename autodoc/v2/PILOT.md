# AutoDoc v2.5 — 파일럿 실행 가이드

MVP(S1~S6) 구현 완료. 이 문서는 실사용 파일럿을 위한 접속·시나리오·확인 방법입니다.

## 1. 접속

| 항목 | 값 |
|---|---|
| **URL** | `https://yuyoung-ai.github.io/YuYoung/autodoc/v2/` (main 배포 후 활성) |
| 로그인 | 기존 BAZ CS 비밀번호 (v1 인증 재사용) · 레벨 1=사용자 / 레벨 2=관리자 |
| 백엔드 | Google Apps Script(v2) + Google Sheets — 이미 연결됨 |
| 설치 | 브라우저에서 "홈 화면에 추가" 시 앱처럼 사용(PWA) |

> **보안 참고**: 현재 GAS 는 `REQUIRE_AUTH=false`(초기 테스트) — 모든 요청이 관리자 권한으로 처리됩니다.
> 사용 범위를 넓히기 전에 스크립트 속성에서 `REQUIRE_AUTH=true` + `AUTH_URL`(v1 인증 /exec)을 설정해 잠그세요.

## 2. 파일럿 시나리오 (여정 F1~F5)

### F1 · 첫 문서 작성 (목표: 3분)
1. **문서 만들기** → 🏆 표시된 양식 선택
2. 폼 입력 — 오른쪽 **미리보기가 실시간**으로 바뀝니다 (생성물과 동일 구조)
3. 필수 항목 100% → **[생성]** → PPT / Excel / PDF 선택 → 파일 다운로드
4. **내 문서**에 이력이 남는지 확인

### F2 · 재사용 작성
- 이어서 작성(임시저장 복원) · 제안 칩(💡 최근 값) 채택 · Undo/Redo(↶ ↷)

### F3 · AI로 회사 규칙 가져오기 (관리자)
1. **관리 → AI로 양식·규칙 가져오기**
2. 문서 종류 선택 → **Prompt [전체 복사]**
3. ChatGPT·Claude·Gemini 등에 붙여넣고 `[회사 문서]` 자리에 실제 문서 내용을 넣어 실행
4. AI가 준 **JSON을 붙여넣기 → [검사]** (설명이 섞이면 "재요청 Prompt 복사"로 회복)
5. 추출된 규칙·용어 확인 → **[승인함에 제출]**

### F4 · 승인 (관리자)
1. **관리 → 승인함** → 항목별 근거 확인 → **승인 / 반려**
2. **학습 상태**에서 **Company DNA 버전이 올라가는지** 확인 (승인분만 반영 — Human Approval)

### F5 · 설정·오프라인
- **설정**: 테마(라이트/다크)·글자 크기 · (관리자) **백업 만들기 / 복원**
- **오프라인 테스트**: 비행기모드로 전환 → 작성·미리보기·생성 계속 → 상단 "오프라인" 배너 → 복귀 시 **자동 동기**(상태바 "동기 대기 N건" → 0)

## 3. 데이터 확인 (관리자)

Google Sheets(v2 DB)의 탭에서 실시간 확인:
- `History` — 생성 이력 · `Drafts` — 임시저장 · `DNA` — 학습 반영(버전 증가)
- `KB` — 용어 · `Learning` — 승인 대기/결정 · `Audit` — 모든 변경(누가·언제·무엇)

## 5. 피드백 수집 포인트

| 항목 | 질문 |
|---|---|
| 속도 | 로그인→첫 다운로드가 3분 이내였나요? |
| 미리보기 | 생성된 파일이 미리보기와 구조가 같았나요? |
| Import | AI JSON 붙여넣기에서 막힘/오류가 있었나요? (오류 코드 캡처) |
| 오프라인 | 연결이 끊겨도 작성이 계속됐나요? 복귀 후 반영됐나요? |
| 용어/문구 | 어려웠던 버튼·용어가 있었나요? |

발견 사항을 알려주시면 보정하겠습니다.
